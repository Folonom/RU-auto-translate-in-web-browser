# RU-auto-translate-in-web-browser
Расширение для Яндекс Браузера (и любого Chromium: Chrome, Edge, Brave), которое переводит весь текст на странице на русский прямо в DOM — посты, комментарии, ленту, всплывающие блоки. Работает там, где другие расширения пасуют: X (Twitter), Reddit и любые сайты с открытым Shadow DOM.
